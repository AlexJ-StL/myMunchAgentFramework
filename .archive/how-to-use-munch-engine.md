# How to use Munch-Engine:



1. **Clone \& Install:** Clone this repo and install the jMunch MCP servers via pip install.

2. **Add Skills:** Drop your .md skill files into the /skills folder.

3. **Initialize Director:** Paste the director-prompt.md into your LLM (Claude, ChatGPT, etc.).

4. **Input Goal:** Give the Director a goal.

- **Example:** "I need to update the API endpoint in my frontend but I don't know where it's defined."

5. **Review ROI:** The Director will find the file, task a Worker to change it, and show you exactly how many tokens you saved compared to reading the whole repo.
